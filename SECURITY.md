# Security Policy

## Supported Versions

We currently support the main branch with security updates.

## Reporting a Vulnerability

We take the security of this project seriously. If you discover a security
vulnerability, please do **not** open a public issue.

Instead, send a private email to security@langstop.com

Please include:

- Description of the vulnerability
- Steps to reproduce
- Affected tools or URLs
- Any potential impact

We will acknowledge receipt within 48 hours and provide a timeline for
resolution. Security issues are our top priority.

## What to Report

- Malicious links in tool submissions
- Phishing or scam websites listed
- Tools that collect data without disclosure
- Libraries with known vulnerabilities
- Spam or deceptive content

## Safe Harbor

We support responsible disclosure. If you report a security issue in good
faith, we will not pursue legal action against you.
