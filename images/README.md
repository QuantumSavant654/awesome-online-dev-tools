# Images

This directory contains screenshots and images for tools listed in the
Awesome Online Dev Tools collection.

## Structure

```
images/
  screenshots/
    tool-name.png    # Tool screenshots (optimized, max 500KB)
  icons/
    category-icon.png  # Category icons
```

## Guidelines

- Screenshots should be PNG format
- Maximum file size: 500KB
- Dimensions: 1280x800 recommended
- Lazy-loaded via GitHub Markdown
- Only include screenshots that add value
